Web Server

A web server is a computer system that stores website files and delivers them to users over the internet when requested by a browser.

When you open a website, the server sends the page to your device using HTTP.

How a Web Server Works

  • 1. A user enters a website URL in the browser.
  • 2. The browser sends a request to the web server.
  • 3. The web server processes the request.
  • 4. The server sends back the requested webpage.
  • 5. The browser displays the webpage to the user.

Web servers work together with browsers using HTTP/HTTPS to deliver web content quickly and efficiently.
