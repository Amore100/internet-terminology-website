HyperText Markup Language (HTML)

HTML is the standard language used to create and structure web pages using elements like headings, paragraphs, and images.

It forms the foundation of all websites.

What is HTML Used For?

HTML (HyperText Markup Language) is used to create the structure of web pages. It tells the browser how content should be arranged on a webpage.

Basic HTML Structure

  • <html> → Root element of a webpage
  • <head> → Contains title and metadata
  • <body> → Contains visible content

Common HTML Elements

  • Headings (h1 to h6)
  • Paragraphs (p)
  • Images (img)
  • Links (a)
