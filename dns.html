Domain Name System (DNS)

DNS (Domain Name System) is a system that translates human-friendly domain names into IP addresses that computers use to identify each other on the network.

How DNS Works

  • 1. User enters a website name (e.g. www.google.com)
  • 2. Browser sends request to DNS server
  • 3. DNS server finds the matching IP address
  • 4. IP address is returned to the browser
  • 5. Browser connects to the web server and loads the site

DNS works like the internet’s phonebook, matching names to numbers (IP addresses).
