🏗️Multitier Architecture

Multitier architecture is a software design approach where an application is divided into separate layers (tiers), each responsible for a specific function. These layers work together to process user requests efficiently.

🧱 1. Presentation Layer This is what the user sees and interacts with (like a browser or app screen). 👉 Its job: Show information and collect user input

⚙️ 2. Application Layer (Logic) This is the “brain” of the system. 👉 Its job: Processes requests and applies rules (decides what happens)

🗄️ 3. Data Layer This is where information is stored. 👉 Its job: Stores and retrieves data (databases)

Each layer performs a specific function to improve efficiency and scalability.
