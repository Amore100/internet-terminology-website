Uniform Resource Locator (URL)

A URL is the address used to locate a resource on the internet, such as a website or file.

A URL consists of three main parts: The Protocol (http://), the Domain Name (example.com), and the Path (/folder/file.html).

Example: https://www.google.com is a URL.
