Transmission Control Protocol / Internet Protocol (TCP/IP)

TCP/IP is a set of communication protocols used to connect devices over the internet. It ensures data is sent correctly between computers.

TCP/IP 4-Layer Model

  • Link Layer: Handles physical transmission of data over hardware like cables and Wi-Fi.
  • Internet Layer: Responsible for IP addressing and routing data between networks.
  • Transport Layer: Ensures reliable delivery of data using TCP or faster delivery using UDP.
  • Application Layer: Provides services like HTTP, FTP, email, and web browsing to users.
