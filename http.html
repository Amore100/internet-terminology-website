HyperText Transfer Protocol (HTTP)

HTTP (HyperText Transfer Protocol) is a communication protocol used to transfer web pages and other data from a web server to a web browser over the internet.

How HTTP Works

HTTP works using a request and response model between the client (browser) and the server.

  • 1. The user enters a URL in the browser.
  • 2. The browser sends an HTTP request to the web server.
  • 3. The server processes the request.
  • 4. The server sends back an HTTP response (web page).
  • 5. The browser displays the web page to the user.

Example

When you type www.google.com, your browser uses HTTP to request Google’s homepage from its server.

Key Features of HTTP

  • It is stateless (each request is independent).
  • It is used for transferring web pages.
  • It works with TCP/IP to ensure data delivery.
