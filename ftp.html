File Transfer Protocol (FTP)

FTP is used to transfer files between computers over a network. It allows uploading and downloading of files from servers.

How FTP Works

  • 1. A user connects to a server using an FTP client.
  • 2. The user logs in with a username and password.
  • 3. Files can be uploaded from the local computer to the server.
  • 4. Files can also be downloaded from the server to the computer.
  • 5. The connection is closed after the transfer is complete.

FTP is commonly used by developers to upload website files to a web server.
